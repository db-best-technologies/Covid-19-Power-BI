Using Powershell to combine CSV files used for COVID-19 Time Series reports

Governments, researchers, and hobbyists are creating dashboards each day for tracking COVID-19 data. Most of these sites are using data located in a Github repository at https://github.com/CSSEGISandData/COVID-19. On March 23, many of the dashboards were showing that all of the sudden indicated that there were no new cases. It turns out that the team maintaining the data announced that they were deprecating three main files dashboards were using. In this blog post, I'm going to show how to use PowerShell to combine CSV files for new time_series_covid19_confirmed_global.csv and time_series_covid19_deaths_global.csv files.
